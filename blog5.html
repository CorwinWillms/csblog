What is NPM?

The fastest way to describe NPM is to say it is the free package manager and installer for Node.js. NPM was created in 2009 as an open source project and today it is the primary way Node.js developers install and manage JavaScript packages.

NPM is still open source. It is the largest open source software registry. In fact, it is the largest software registry period. In 2018 the NPM registry achieved over a billion downloads in a single day. There are over 800,000 packages available for download through NPM.

NPM is very easy to use. It is installed after installing Node using the following command: 

                    
                    
                        npm install -g npm

After installing the latest version of NPM, you can install a package in the NPM registry with: 

                    
                    
                        npm install PACKAGE-NAME
